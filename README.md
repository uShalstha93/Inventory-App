
# Inventory Management System

Tech Stack:
(MERN)
- React JS
- React Router
- Redux
- React Bootstrap
- Formik & Yup for Form Validation in React
- MongoDB
- Node JS
- Express JS
- JsonWebToken Authentication
- React Bootstrap Toast for Alert
